# PLACE
‘Place’앱은 Google map API를 이용하여 자신의 위치 뿐만 아니라 장소 검색기능과 마커를 생성하여 저장할 수 있습니다. 또한 친구목록에 친구들과 그룹을 생성하여 마커를 공유할 수 있습니다.

1. 지도 검색 기능 : Google Map api를 이용하여 구글 지도 내 검색 및 현재 자신의 위치를 알 수 있습니다.

2. 지도 내 마커 표시 : 원하는 지점에 마커를 지정하여 나만의 메모를 저장할 수 있습니다.

3. 공유 기능 : 사용자들끼리 그룹을 생성하여 저장한 마커를 공유할 수 있습니다.
